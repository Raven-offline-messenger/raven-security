# Research

This folder hosts notes, write-ups, and collaboration material
that supports the public security story but is not part of the
authoritative specification.

Suitable content:

- Comparison studies (Raven vs other mesh / privacy systems).
- Discussion notes from external reviewers.
- Drafts of future protocol additions (e.g. distance bounding,
  cover traffic) that are not yet committed to the roadmap.
- Bibliographic references.

Material in this folder should be treated as **draft / informal**
unless explicitly promoted into the top-level documents.

If you would like to contribute research, please open an issue or
email `security@raven-messager.com` first. We do not currently
accept arbitrary pull requests into the spec docs.
