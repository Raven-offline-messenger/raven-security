# Audits

This folder is reserved for third-party security audit reports.

**Status (May 2026):** empty.

Stage 5 of the [Trust Roadmap](../TRUST_ROADMAP.md) commits us to
commissioning independent review of the ATSAM cryptographic core
and the reference implementation. When audit reports arrive, they
will be published here — including unflattering findings.

When that happens, expect the layout:

```
audits/
  YYYY-MM-firm-name/
    report.pdf           original auditor report
    summary.md           plain-English summary
    findings/
      F-001-<name>.md    one file per finding, with fix status
```

Until that day, this README is the entire content of the folder.
We will not pretend to have completed an audit until we have.

For the planned audit timeline, see
[`../TRUST_ROADMAP.md`](../TRUST_ROADMAP.md).
